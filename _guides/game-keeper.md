# Game Keeper

Game Keepers maintain game systems and coordinate rewards and content.

Responsibilities

- Review and approve submitted game content and rewards.
- Assist DMs with timing, pacing, and problem-solving during games.
- Help finalize rewards and support players in claiming them.

Collaboration

- Work closely with DMs and Trackers to ensure events and bounties run smoothly.
